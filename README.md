# 🩸 Blood Donation System – Backend API

## 📌 Overview

This is the backend service for the Blood Donation Web Application, built using the PERN stack (PostgreSQL, Express, React, Node.js).

The API handles user authentication, donor management, blood requests, and admin operations. It is designed to support a scalable and secure system for connecting blood donors with recipients.

--

## 🚀 Features

- 🔐 User Authentication (JWT + HTTP-only cookies)
- 👤 Donor Registration & Profile Management
- 🩸 Blood Donation Requests
- 📍 Location-based donor filtering
- 🛡️ Role-based Access Control (Admin & Users)
- 📦 RESTful API architecture
- 🧾 Input validation & error handling

--

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL
- JWT Authentication
- bcryptjs
